Interactive Visual Guide

How Machine Learning
is built from Mathematics

A 19-chapter visual journey through Linear Algebra, Statistics, Probability, and Calculus — one story, one dataset, from raw data to a fully trained classifier.

🏦 One usecase, all 19 chapters:  Should the bank approve this loan application?
No prior ML knowledge needed. Each chapter builds on the last. If you want to practice the row operations yourself, visit the REF Practice tool ↗ alongside this guide.

Act 1 — Linear Algebra (Ch 1–7)

Structure loan data as matrices, make predictions, and solve for optimal parameters.

Chapter 1  Start here

Chapter 5  Builds on Ch 4
Rank & Linear Dependence

Chapter 6  Builds on Ch 4 & 5
Gradient Descent
Matrix inversion (Ch 4) is too slow at scale and breaks when features correlate (Ch 5). Watch gradient descent find the same θ by rolling down the loss surface — no inversion needed.

Chapter 7  Builds on Ch 5
PCA — Finding the Best Axes
Income and age overlap (Ch 5). PCA compresses them into one axis using eigenvectors of XᵀX. Step through centring → covariance → eigenvectors → 2D→1D projection.
