Prediction = Matrix Multiplication

In Ch 2 we saw ML needs to minimize error, which means we need to compute predictions first. For any applicant, a prediction is a single dot product — one row of X times the θ vector. For all 5 applicants simultaneously, it's Xθ = ŷ. This is why matrices matter.

Key insight: ŷ = Xθ is a (5×3)·(3×1) = (5×1) matrix multiplication. Changing any weight θᵢ shifts all predictions at once — because every row of X uses all three weights.
The remaining question: We tried random θ values — but how do we find the best θ automatically? That's exactly what the Normal Equation in Chapter 4 solves.
